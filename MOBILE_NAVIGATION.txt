CUSTOMIZATION
=============

Want to change menu width?
In styles.css, find:
    width: 70%;
    max-width: 300px;

Change to:
    width: 80%;  (wider)
    width: 60%;  (narrower)

Want to change breakpoint?
In styles.css, find:
    @media (max-width: 768px)

Change to:
    @media (max-width: 1024px)  (activate on tablets too)
    @media (max-width: 480px)   (only tiny phones)

Want to change colors?
Menu background:
    background: #003d82;  (current)

Overlay:
    background: rgba(0, 0, 0, 0.5);  (50% black)
